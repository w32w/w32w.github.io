常规手法失效的前提下，在博客上显示pdf。笨办法。

PDF

为了存档跑团的log而想要显示pdf，不知道为什么，网上的方式对我而言都行不通……折腾了一会，选择了一种弯弯绕绕的方式

Next主题配置

themes/next/config.yml ，跟着一些网上教程，一步步修改下来

本地文件路径

我放在了 .\blogs\themes\next\source\lib\pdf\web 下

引用pdf

搜索后发现上传的文件在 github 仓库里的 w32w.github.io/lib/pdf/web/ 下，使用 {% pdf https://w32w.github.io/lib/pdf/web/insane.pdf %} ，可以正常显示。

也可以使用 <object data="https://w32w.github.io/lib/pdf/web/insane.pdf" type="application/pdf" width="100%" height="700px"> ，可以自己调整高度，蛮不错（参考）

示例：

总结

  • 随便把pdf丢在一个地方上传
  • 去自己的仓库目录下，搜索这个Pdf，获取直链（以及显示规律）
  • 使用 {% pdf https://[...].pdf %} 或者 <object data="https://[...].pdf" type="application/pdf" width="100%" height="700px"> 来插入正文
